## Objectives

Analyze a retail company’s sales data using **MySQL**.
The data was originally in Excel files, then **imported into MySQL** for cleaning and analysis, and finally **exported back to Excel** to build a professional dashboard.

---

## Tools

* **MySQL** → Data cleaning, transformation, and analysis
* **Excel** → Dashboard creation and visualizations
* **GitHub** → Project documentation and storage

---

## Steps

1. **Explored the dataset in Excel**

   * Reviewed the data files to understand structure and content.

2. **Created the MySQL database**

   * Set up a database to store the retail sales data.

3. **Imported Excel data into MySQL**

   * Converted Excel files into tables: `customers`, `products`, `sales`.

4. **Structured the tables**

   * Defined proper data types and constraints for clean, organized tables.

5. **Performed analysis using SQL**

   * Calculated key business metrics:

     * Total revenue
     * Total number of sales
     * Sales by product
     * Sales by customer
     * Total number of customers
     * Total number of products

6. **Built the Excel dashboard**

   * Summarized insights with:

     * **Total Customers**
     * **Total Products**
     * **Top Customers** (highest-spending)
     * **Top Products** (best-selling)
